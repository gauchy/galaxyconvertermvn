How to run:
run mvn package on the top directory
cd target
java -cp galaxyconverter-1.0-SNAPSHOT.jar com.thoughtworks.galaxyconverter.client.Client


(Requires java7)


Input fomat information:

Please follow these format
Input:To give mapping like 'glob is I' enter glob=I
Question:To get value of a galactic number like 'how much is pish tegj glob glob' enter 'pish tegj glob glob=?' 
Input:To give item value input like 'glob glob Silver is 34 Credits' enter 'glob glob=Silver=34'
Question:To get value of an itme like 'how many Credits is glob prok Silver' enter 'glob prok=Silver=?'
Exit:exit
